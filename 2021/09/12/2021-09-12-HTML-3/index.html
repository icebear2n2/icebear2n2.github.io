콘텐츠 모델



MetaData

콘텐츠의 style, script을 설정하거나 다른 문서와의 관계 등의 정보를 포함하는 요소

태그 안에 들어간다

Flow

문서의 사용되는 대부분의 요소

Sectioning

헤딩과 푸터의 범위를 결정하는 요소, 아웃라인이 있다

Heading

섹션의 헤더를 정의하는 요소

Phrasing

문서의 텍스트이며 문단 내부 레벨로 마크업 하는 요소

Embedded

이미지, 비디오, 플래시 등 외부 콘텐츠를 문서 내에 표현하는 요소

Interactive

사용자와 상호작용을 하는 요소

시멘틱 마크업

컴퓨터(브라우저)가 잘 이해할 수 있는 코드

의미에 맞는 요소를 사용

문서의 구조화

인간과 기계가 모두 이해할 수 있는 것이 목표

Block & Inline Level

Block Level

한 줄에 하나의 요소 표시

일반적인 모든 요소 포함하며, divm h1 ~ h6, p, ul, li, table 등

Inline Level

한 줄에 여러 개의 요소 표시

block 레벨의 자식요소이며, span, i, img, em, strong 등

는 인라인 레벨 요소이지만 자손으로 블록 레벨 요소를 가질 수 있다
