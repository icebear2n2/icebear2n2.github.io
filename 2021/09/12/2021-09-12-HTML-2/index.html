List

    unordered list로 순서가 없는 리스트

    OL

      ordered list로 순서가 있는 리스트

      DL

      description list로 용어를 설명하는 리스트
      definition term로 용어를 구분
      definition description로 용어의 정의를 나타낸다

      리스트의 중첩

      이미지(Image)

      문서에 이미지를 삽입

      src : 이미지 경로를 지정

      상대 경로 : 현재 위치, ./

      절대 경로 : c:/, http://

      alt : 이미지의 대체 텍스트를 입력

      width/height : 이미지의 크기를 지정

      GIF VS JPG VS PNG

      gif : 256색 제한, 용량 적음, 애니메이션과 투명 이미지 가능

      jpg : 높은 압출률과 자연스러운 색상 표현, 사진이나 일반적인 그림에 사용, 투명 X

      png : jpg와 비교했을 떄 이미지 손실 x, 투명 . 반투명 둘 다 지원

      테이블 구조 관련 태그

      : 표의 제목

      : 제목 행을 그룹

      : 바닥 행을 그룹 : 본문 행을 그룹

      테이블 관련 속성

      colspan: 셀을 가로 방향으로 병합

      rowspan: 셀을 세로 방향으로 병합

      폼 관련 요소 (Form)

      Input 요소

      대표적인 폼 요소로, 다양한 type 속성으로 여러 종류의 입력 양식으로 나타난다

      text, password, radio, checkbox, file, image, submit, reset, button . . .

      text

      radio

      submit

      image

      SELECT 요소

      Textarea 요소

      여러 줄 텍스트 입력 상자

      Button 요소

      사용자가 클릭 가능한 버튼

      submit, reset, button

      Label 요소

      폼 컨트롤과 연결 시켜주기 위함으로 웹 접근성 향상에 도움이 된다 (필수적)

      Fieldset, Legend 요소

      : 여러 개의 폼 요소를 그룹화하여 구조적으로 만들기 위해 사용 : 폼 요소의 제목으로
      요소 내부에 작성

      Form 요소

      폼 데이터를 그룹화하여 서버에 전송

      action : 폼 데이터를 처리하기 위한 서버의 주소

      method : 데이터를 전송하는 방식을 지정

      • get(민감한 정보 X), post
