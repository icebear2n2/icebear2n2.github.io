tmux 명령어 정리

2021-07-28 Linux PV:


설치

MAC OS

Ubuntu

tmux는 각 터미널 인터페이스를 세션, 창의 세 가지 범주로 구분한다
여러 인터페이스를 세 개의 다른 레이어로 분리함으로써 작업 공간을 더 잘 구성할 수 있다

tmux 명령어

<Ctrl-b> " : 가로로 분할

<Ctrl-b> % : 세로로 분할

<Ctrl-b> 방향키 : 원하는 창으로 이동

<Ctrl-b> ; : 원하는 창으로 이동

<Ctrl-b> <Ctrl-o>: 창 위치 회전

<Ctrl-b> q: 창 번호 표시

<Ctrl-b> x : 창 닫기

<Ctrl-b> : : 창 사이즈 조절
resize-pane -D 10: 창의 크기를 10셀 아래로 조정
resize-pane -U 10: 창의 크기를 10셀 위로 조정
resize-pane -L 10: 창 크기 조정은 10개의 셀을 남겼습니다.
resize-pane -R 10: 창의 크기를 오른쪽 10셀로 조정
resize-pane -t 2 30: id 2의 창 크기를 30셀 아래로 조정

<Ctrl-b> c : 새 창 만들기
tmux new-window -n <이름> : 이름이 있는 새창 만들기

<Ctrl-b> n : 다음 창으로 이동

<Ctrl-b> p : 이전 창으로 이동

<Ctrl-b> <0-9> : 창 번호로 창 이동

tmux next-window : 다음 창으로 이동

tmux previous-window : 이전 창으로 이동

tmux last-window : 창 번호로 창 이동

<Ctrl-b> w : 목록으로 창 이동

tmux list-windows : 목록으로 창 이동

<Ctrl-b> , : 창 이름 변경

tmux rename-window play : 창 이름 변경

<Ctrl-b> f : 윈도우 검색

<Ctrl-b> & : 창 닫기

tmux kill-window 현재 창 닫기
