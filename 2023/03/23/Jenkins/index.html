Jenkins

Jenkins

젠킨스(Jenkins)는 오픈 소스 자동화 도구로, 개발자들이 소프트웨어 개발 생산성을 높일 수 있도록 해줍니다.



CI - Continuous Integration

Jenkins는 CI/CD 파이프라인에서 중요한 역할을 합니다.

💡 Continuous Integration이란,

여러 개발자들이 동일한 코드 베이스에서 작업하고 있는 경우,
코드 변경 사항을 자동으로 빌드하고 테스트하여 문제를 빠르게 파악하고 해결할 수 있는 개발 방법

이를 위해 Jenkins는 코드를 체크아웃하고 빌드 / 테스트 / 배포 등의 작업을 수행하는 파이프라인을 구축할 수 있습니다.

CD - Continuous Deployment

젠킨스는 또한 Continuous Deployment를 지원합니다.

💡 Continuous Deployment란,

코드 변경 사항이 검증되면 자동으로 프로덕션 환경으로 배포되도록 하는 개발 방법

Jenkins를 사용하면, 배포를 자동화하여 애플리케이션 배포 시간을 단축하고, 배포 과정에서 발생할 수 있는 인적 오류를 최소화할 수 있습니다.



Plugins

Jenkins는 다양한 플러그인을 제공하여 유연하게 확장할 수 있습니다.
이러한 플러그인들은 빌드, 테스트, 배포 등의 작업을 자동화하기 위한 다양한 기능을 제공합니다.
또한 Jenkins는 다양한 소스 코드 관리 시스템과 통합이 가능합니다.

이를 통해 개발자들은 자신들이 선호하는 도구를 사용하여 작업할 수 있습니다.
